Private Codespace Instance

Private Proxy

Run this Codespace and set the target upstream you want to expose privately.

Using the proxy

After configuration, your proxy base path is /p/. Example: /p/path/to/resource will be forwarded to the configured upstream.

Note: Admin configuration requires ADMIN_TOKEN to be set as an environment variable on the server process (not only in this UI).
