Private Proxy — Super Easy Admin Setup

Open this page in your Codespace preview. If ADMIN_TOKEN is not set, follow the simple steps below.

Step 1 — Create admin token (one-click)

You can generate a secure token in your browser or paste your own. This token will be written to .env on the server (DO NOT commit).

After saving, admin actions are enabled immediately without restart.

Using the proxy

After configuration, your proxy base path is /p/. Example: /p/path/to/resource will be forwarded to the configured upstream.
